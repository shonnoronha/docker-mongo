## Docker Express App

### To Run The App Locally

``` sh
git clone {url}
cd dock-express-demo/
git install
npm start
```

### To Run The App in Docker Container

```sh
git clone {url}
cd dock-express-demo/
docker compose up -d
```

Visit [localhost:3000](http://localhost:3000) To See The App Running!
